Regular Expressions (Regex)

More technical interview questions

What are regular expressions?

They are strings and characters used as a search pattern which has multiple uses such as for searching text, matching certain words and values, or replacing text.

How do you use regular expressions?

Regex characters are placed in diagonal brackets '/regex characters/'. There are two main methods to test expressions with

1) .test()

This method will return a boolean. First, you will define a set of characters you want to match

let regexEx = /char/

Then, use the test method on regexEx, it will return true if characters match, false if they dont

regexEx.test('character')

The above example will return true since 'char' is within the 'character' argument.

2) .match()

This take an expression to match as an argument, and will return an array with the first value being the match found. There are multiple expressions that can be used, check out regex101 to see different expressions which can be used and practice with them.

In the following example, a 'wildcard' is used which uses a period (.) to match any single character.

let regexEx = /./
let stringEx = 'character'
stringEx.match(regexEx) //[ 'c', index: 0, input: 'character', groups: undefined ]

The above example returns an array, where the first value will be the value returned, which in the above case is 'c' because the period matches the first character and only that character.
